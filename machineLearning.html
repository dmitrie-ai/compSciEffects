Machine Learning

Here are just a few applications of machine learning that are interesting:

  • The heavily hyped, self-driving Google car? The essence of machine learning.
  • Online recommendation offers such as those from Amazon and Netflix? Machine learning applications for everyday life
  • Knowing what customers are saying about you on Twitter? Machine learning combined with linguistic rlie creation.
  • Fraud detection? One of the more obvious, important uses in our world today.

As you can see, there are many applications of machine learning. To learn how to use it there are numerous programming tools such as:

  • Keras,
  • Tensorflow,
  • Pytorch,

each of which can be used from python.
